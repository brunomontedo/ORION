# ORION
Sistema inteligente para inspección precompra de vehículos.

Características principales:

- Inspección guiada mediante IA
- Vehículo 3D interactivo
- EOIV (Estándar ORION de Inspección Vehicular)
- ARGO (Asistente IA)
- Historial técnico por vehículo
- Trust QR
- Marketplace de inspectores
